# 知識守護隊

小一到小六的 3D 問答塔防遊戲。玩家答題取得能量，在 3D 浮島上建造知識塔，抵擋迷霧怪、粗心怪與時間裂影。

## 特色

- 原創 Three.js 低多邊形 3D 戰場
- 程式合成背景音樂與音效，沒有外部音檔授權問題
- 題庫涵蓋小一到小六、上下學期、期中/期末、國語/英文/數學/自然/社會
- 題庫已擴充到 2680 題；單一條件全科約 110 題以上，單科約 22 題以上
- 開局前會列出目前條件下各科題量與能力覆蓋，方便確認題庫分布
- 本局目標會顯示答題進度、剩餘題數、即時正確率與已看過題數，支援 20 到 30 題長局
- 答錯會顯示正確答案、解題與提示，錯題會回到題庫重複練習
- 弱點回補會指出目前最需要修復的科目與能力、待修復題數與回補進度
- 本局能力雷達會即時顯示五種學習能力的正確率、錯題與修復數
- 錯題複習佇列、掌握度、壓力值與連擊獎勵會直接影響戰局
- 答對特定科目會啟動同科塔的短暫共鳴，提升傷害與射速
- 建塔前可選前線、快速、弱點、厚盾四種目標策略，讓同一座塔有更多戰術變化
- 下一波敵群預告、塔種策略說明、任務目標與最近戰績支援長期打磨
- 提示可在主畫面使用，答題後保留完整正解與解題回饋
- GitHub Pages 自動部署

## 開發

```bash
pnpm install
pnpm dev
```

## 建置

```bash

```
